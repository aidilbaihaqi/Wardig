# WARDIG (Warisan Digital) - Project Brief
# Platform Digital untuk Produk Lokal UMKM Sungai Enam

## OVERVIEW
Platform digital yang menghubungkan produk lokal UMKM Sungai Enam dengan cerita di baliknya melalui QR Code, menciptakan emotional connection dan mendukung ekonomi lokal.

## TARGET AUDIENCE
- Primary: Masyarakat awam yang ingin mengetahui info produk, cerita pembuatan, sejarah, dan nilai budaya
- Secondary: Wisatawan dan pecinta produk lokal

## ADMIN
- Single admin: Petugas kelurahan Sungai Enam yang mengurus UMKM di daerah tersebut

## TECH STACK
- Backend: Laravel (PHP)
- Frontend: React + Inertia.js
- Styling: Tailwind CSS
- Database: MySQL
- QR Generation: PHP QR Code Library

## WEBSITE STRUCTURE

### ADMIN PANEL (/admin/*)
1. Dashboard Analytics
   - Total produk terdaftar
   - Total scan QR (harian/bulanan)
   - Produk paling populer
   - Grafik aktivitas scan
   - Recent reviews

2. Manajemen Produk (/admin/products)
   - List produk dengan preview QR
   - Form tambah/edit produk (nama, deskripsi, sejarah, filosofi)
   - Upload video (max 2 menit, MP4)
   - Upload galeri foto produk
   - Generate QR Code (500x500px, PNG, dengan logo)
   - Download QR Code

3. Manajemen UMKM (/admin/umkm)
   - Database penjual/pembuat
   - Profil lengkap (nama, alamat, kontak, cerita)
   - Assign produk ke UMKM

4. Review Management (/admin/reviews)
   - Moderasi review customer
   - Approve/reject review
   - Response ke review

### PUBLIC PAGES (Diakses via QR)
1. Product Landing (/product/{unique_code})
   - Hero section dengan nama produk & foto utama
   - Video dokumenter (autoplay muted, controls)
   - Quick info (pembuat, lokasi)
   - Navigation ke section lain

2. Product Story (/product/{unique_code}/story)
   - Sejarah produk
   - Proses pembuatan (step-by-step)
   - Filosofi & nilai budaya
   - Galeri foto proses

3. Maker Profile (/product/{unique_code}/maker)
   - Profil UMKM/pembuat
   - Lokasi & kontak
   - Cerita di balik usaha
   - Produk lainnya

4. Gallery (/product/{unique_code}/gallery)
   - Galeri foto produk
   - Behind the scenes
   - Zoom & lightbox functionality

5. Reviews (/product/{unique_code}/reviews)
   - Customer reviews & ratings
   - Form tambah review (nama, rating, komentar)
   - Photo reviews (optional)

## DATABASE SCHEMA

### umkm_profiles
- id (Primary Key)
- name (VARCHAR)
- owner_name (VARCHAR)
- address (TEXT)
- phone (VARCHAR)
- email (VARCHAR)
- story (TEXT)
- established_year (YEAR)
- logo_path (VARCHAR)
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

### products
- id (Primary Key)
- umkm_id (Foreign Key)
- name (VARCHAR)
- description (TEXT)
- history (TEXT)
- philosophy (TEXT)
- video_path (VARCHAR)
- qr_code_path (VARCHAR)
- unique_code (VARCHAR, UNIQUE)
- status (ENUM: active, inactive)
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

### product_images
- id (Primary Key)
- product_id (Foreign Key)
- image_path (VARCHAR)
- alt_text (VARCHAR)
- sort_order (INTEGER)
- is_featured (BOOLEAN)
- created_at (TIMESTAMP)

### qr_scans
- id (Primary Key)
- product_id (Foreign Key)
- scanned_at (TIMESTAMP)
- ip_address (VARCHAR)
- user_agent (TEXT)
- location_data (JSON, optional)

### reviews
- id (Primary Key)
- product_id (Foreign Key)
- customer_name (VARCHAR)
- rating (INTEGER, 1-5)
- comment (TEXT)
- review_images (JSON, optional)
- status (ENUM: pending, approved, rejected)
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

### users
- id (Primary Key)
- name (VARCHAR)
- email (VARCHAR, UNIQUE)
- password (VARCHAR)
- role (ENUM: admin)
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

## TECHNICAL SPECIFICATIONS

- Custom logo di tengah QR

- Format: MP4
- Maximum duration: 2 menit

- Autoplay muted dengan controls

- Geographic data (jika available)

- Time-based analytics (daily/monthly)
- Review statistics
